# Game Performance & Revenue Analysis

## Project Overview
This project analyzes game performance using Google Sheets by calculating key product metrics such as revenue, conversion rate (CR), ARPPU, and user demographics.

## Tools Used
- Google Sheets
- Data Analysis
- KPI Calculation

## Key Metrics
- Total Revenue
- Paid Users Count
- Conversion Rate (CR)
- ARPPU
- User Age Statistics

## Insights
- Game 3 has the highest ARPPU despite a lower conversion rate.
- Game 1 has a stronger conversion rate but lower revenue per paying user.
- User demographics may influence purchasing behavior.

## Recommendations
- Improve onboarding to increase conversion rates.
- Test pricing and in-game offers.
- Focus on high-value users to maximize revenue.
